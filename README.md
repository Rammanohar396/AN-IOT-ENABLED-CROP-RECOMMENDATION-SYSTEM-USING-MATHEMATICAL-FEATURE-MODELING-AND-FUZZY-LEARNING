## 📌 Project Overview

 AN-IOT-ENABLED-CROP-RECOMMENDATION-SYSTEM-USING-MATHEMATICAL-FEATURE-MODELING-AND-FUZZY-LEARNING is a final year B.Tech project at **Vel Tech Rangarajan Dr. Sagunthala R&D Institute of Science and Technology, Tamil Nadu**. It provides real-time, data-driven crop recommendations by combining IoT sensor inputs with ensemble machine learning and fuzzy logic — while also calculating carbon footprint and carbon credit potential.

---

## ✨ Features

| Feature | Description |
|---|---|
| 🌾 Crop Recommendation | Ensemble ML predicts best crop from 9 parameters |
| 🧪 Fuzzy Fertilizer Advice | NPK deficiency + dosage via fuzzy inference |
| 💧 Irrigation Assessment | Water quality and moisture analysis |
| 🌍 Carbon Footprint | IPCC-based CO₂e estimation per crop/season |
| 💰 Carbon Credit Estimator | Voluntary market credit potential ($15/tonne) |
| 📊 Performance Dashboard | Chart.js model accuracy visualizations |
| 📥 Download Report | One-click HTML report generation |
| 📱 Responsive Design | Desktop, tablet, and mobile ready |

## 🛠 Tech Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript (ES6+)
- **Charts**: Chart.js 4.4.1
- **Fonts**: Google Fonts — Syne + DM Sans
- **ML**: Rule-based scoring engine (JavaScript simulation)
- **Fuzzy Logic**: Triangular membership functions (JavaScript)
- **Carbon**: IPCC AR6 emission factors
- **Hosting**: GitHub Pages
